About me

I am a 4th-year student who has a dream to become a game programmer and make something special and awesome for the game industry. I have fallen in love with games since I was a kid and from that my love and passion for games becomes stronger and stronger. I started my journey to game programming when I studied in Vietnamese-German university (Bachelor of Computer Science) with C++, UE and have made some projects with that. Now I am ready to make my dream come true.

FPS Zombie Game

FPS Zombie Game is a project that I did by my self in C++ and Blueprint. It is a short FPS project that you need to kill all the zombies to survive and win the game.

Here are some of the basics features of the project:

  • Damage different between head and body
  • Ammo system
  • Wave-based mechanic

Crypt Raider

Crypt Raider, is a small project that need to pick up objects and place objects to the right places to take the treasure out. I made this project to learn about the line tracing and overlap event in UE, using C++ and Blueprint.

  • Line tracing
  • Overlap event
  • Lumen light system

Toon Tanks

Toon Tanks, is a project that player will control a tank and try to destroy all the turrets before being destroy. I learned how to make input system and also some basic components of UE.

Living Room

I made this project to learn some of the basic functions and features of Blender.
